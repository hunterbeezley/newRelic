# newRelic

A collection of scripts for use within New Relic.

## Scripts

- **AccountDeletionPrep** - Automates deletion of alert policies, channels, and destinations when preparing accounts for deletion
- **auto-tsv** - Automates TSV (Technical Support Verification) processes
- **bulk_deployment_markers** - Creates deployment markers for multiple application GUIDs from a CSV file
- **Bulk-user-update** - Updates email addresses for multiple users via the User Management API
- **compare.py** - Utility for comparing data sets
- **EventTypesBillable** - Analyzes billable event types in New Relic accounts
- **parent_accounts.py** - Manages parent account relationships
- **scim-delete** - Deletes SCIM users by email lookup via the SCIM API
- **SCIM-Delete-main** - Automates SCIM user ID lookup and deletion
- **scimControlCenter** - Interactive tool for managing SCIM users (fetch, update, delete)
- **userDeleteOneMonth** - Filters and deletes user records older than one month
